# CS 4375 Project 3: Deep Learning for MNIST and CIFAR-10

This project implements and evaluates Multilayer Perceptrons (MLPs) and Convolutional Neural Networks (CNNs) on the MNIST and CIFAR-10 datasets using PyTorch.

## Files
- `ML_Project_3.ipynb` — main notebook with all code
- `final_mlp_retrained_results.csv` — final MLP test results
- `final_cnn_retrained_results.csv` — final CNN test results
- `all_final_results.csv` — combined final results
- tuning CSV files — saved hyperparameter tuning results
